Welcome to the
Training Resource Execution Kit
Under Interim Authority To Test  / BETA Release

The Training Resource Execution Kit (TREK) is a repository of data, information, exercise content, training support packages and vignettes from exercises. In addition to the TREK being an automated exercise design tool, the tool kit will allow HUMINT & CI trainers to create, store, and reuse training packets containing required forms, reports, and associated products (i.e., Detainee Packet, CI Case Folder, and Source Dossier). The application / tool will allow trainers to quickly develop vignettes, storylines, supporting events, roles, and depict a simplified human network, all within the framework of an overarching scenario and tailored training objectives.

Exercise
Design

Provides an automated tool to design exercises as outlined by TRADOC's Exercise Design Guide (TC-7-101).

Data
Repository

Provides a means to not only store data but an application that will leverage the ability to reuse the stored data.

Search

Ability to search for whole training packets or by specific elements of a packet (i.e., vignettes, storylines, events, roles, missions, etc.) based on specific requirements.

Feedback

To avoid common mistakes and allow for duplicating best practices, the tool will also allow for feedback on existing training packets.
